# REST API WITH JWT AUTHENTICATION

This project is a simple REST API built using Core PHP and MySQL with JWT (JSON Web Token) based authentication.
It provides secure user login and protected routes using token-based access and API testing is done with the help of Postman.

## Features

- MVC Architecture
- JWT Authentication
- Middleware Protection
- Password Hashing
- Environment Variables
- REST API Routing
- CRUD Operations

## Technologies Used

- PHP
- MySQL
- Apache
- JWT
- Postman

## API Endpoints

### Authentication

POST /api/register
POST /api/login

### Patients

GET /api/patients
POST /api/patients
PUT /api/patients/{id}
DELETE /api/patients/{id}

## Security Features

- JWT Token Validation
- Password Hashing
- Protected Routes
- Prepared Statements
